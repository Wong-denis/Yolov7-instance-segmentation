HOW TO MAKE COCO JSON FOR DATASET(use main.py):
    $ python3 main.py --dataset new_dataset/ --output_file coco_json/output_file_path.json
    (dataset should be a folder, template_file: default to be "annot_template_coco.json")
    (max_num: maximum possible #contours, min_area: minimum possible area of contours)
	output_file is default to save in coco_json/ so no need to type dir.
min_area:
	resize_new_1108/ : 2150
	crop_new_1108/ : 5000
